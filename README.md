0. Clone the repository locally.
1. First of all, for install all dependencies type in console "npm install".

2. For compile all .scss files into .css type in console "npm run compile". All .scss files will compile to "src/dist" folder and styles will automatically apply to all pages.

3. After that you may use "Live Server": open "index.html" from root of "src" folder, click on "Go Live"- button.

4. You may navigate through project clicking on:
   1. nav-menu references in header ("Homepage", "Catalog", "About Us", "Contact Us");
   2. icon "cart" in header;
   3. buttons "View All Items", "View Product", "Add To Cart", "See All Models".
   4. For LogIn you must click on icon "user" in header.
